# Tipe Input

Tag `<input>` tersendiri merupakan tag multifungsi. Tag tersebut dapat menerima _input_ teks, angka, tanggal, _checkbox_, pilihan tunggal, dan lain-lain dari pengguna. Kamu dapat menjelajahi semua tipe input di [sini](https://www.w3schools.com/html/html_form_input_types.asp).

Untuk menentukan tipe input, kita bisa menggunakan atribut `type`.

### Contoh

```html
<input type="number" />
```

### Tantangan

Tambahkan atribut `type` dengan value `text` ke tag `<input>` yang tersedia.
