# HTML `lang`

Agar website kita lebih aksesible bagi browser, kita bisa menambahkan atribut `lang` ke tag `<html>`. Hal ini untuk memberi tahu browser terkait bahasa yang digunakan dalam konten _website_ kita.

### Contoh

```html
<html lang="en">
  <head>
    <!-- Konten metadata -->
  </head>
  <body>
    <main>
      <!-- Konten utama -->
    </main>

    <footer>
      <!-- Isi Footer -->
    <footer>
  </body>
</html>
```
