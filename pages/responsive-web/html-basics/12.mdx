# Tag Sebagai Link

Kita sudah membuat tag `<a>` berada di tag lain. Namun, taukah kamu bahwa kita bisa mengubah tag lain menjadi _clickable_ (dapat di-klik) dengan membungkusnya di dalam tag `<a>`.

### Contoh

```html
<a href="https://ngoding.org">
  <img src="https://contoh.com/ini-contoh-link-gambar.jpg" alt="Contoh" />
</a>
```

### Tantangan

Buat gambar kucing yang ada menjadi _clickable_ dan arahkan ke `https://freecatphotoapp.com` di tab baru.
