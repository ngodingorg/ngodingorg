# Link

Supaya _website_ kamu punya struktur yang baik, kamu bisa menautkan halaman _website_ lain. Yup, seperti yang sudah kamu bayangkan ada tag khusus untuk hal itu. Tag ini disebut dengan _anchor tag_ atau `<a>`.

Untuk menggunakan tag `<a>`, kamu bisa menambahkan "isi" yang dibungkus di dalam `<a>` dan `</a>`. Serta atribut `href` sebagai tujuan dari link tersebut.

### Contoh

```html
<a href="https://ngoding.org">Buka ngoding.org</a>
```

### Tantangan

Buat link di bawah `p` bertuliskan "**Buka Galeri**" dan tujuan-nya adalah `https://freecatphotoapp.com`.
