# Tag `figure`

Di dalam HTML, sering kali kita ingin menyatukan suatu gambar dengan deskripsi gambar tersebut. Kita bisa saja menambahkan tag `<p>` di bawah gambar. Namun, HTML sudah menyediakan tag yang membantu kita mencapai hal itu.

Tapi sebelum itu, mari kita berkenalan dengan tag `<figure>`. Tag figure ini merupakan tag yang nantinya akan menyatukan gambar dengan deskripsi gambar tersebut.

### Contoh

```html
<figure>
  <img
    src="https://contoh.com/ini-contoh-link-gambar.jpg"
    alt="Ini contoh gambar"
  />
</figure>
```

### Tantangan

Tambahkan tag `<figure>` di gambar susu yang kita buat sebelumnya.
