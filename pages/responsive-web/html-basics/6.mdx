# _Image_ Tag

Kita ingin membuat aplikasi untuk koleksi foto kucing kamu. Tapi, bagaimana cara menampilkan gambar? Yup, seperti yang kamu bayangkan, kita bisa menggunakan tag yang sudah ada. Tag itu adalah `<img>` tag.

Tapi, ada yang unik, nih. Tag untuk gambar tidak punya tag penutupnya ğŸ˜±. Tag seperti ini disebut dengan _self-closing tag_.

### Contoh

```html
<!-- Atau dapat dituliskan: <img> -->
<img />
```

### Tantangan

Tambahkan tag gambar baru di bawah paragraph tag (tenang aja, gambarnya memang belum muncul kok).
