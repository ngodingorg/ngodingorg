# `action` dalam `form`

Di dalam sebuah formulir, kita perlu mengirimkan isian data yang sudah kita isikan ke suatu tempat pengumpulan data. Selayaknya dalam pemilihan umum, seluruh surat suara dikumpulkan untuk dihitung.

Di dalam `<form>` HTML, kita dapat memberi tahu kode kita bahwa data isian `form` ingin kita lakukan sesuatu (misalnya ingin kita kirim) dengan atribut `action`.

### Contoh

```html
<form action="/url-untuk-upload-data">
  <!-- Isian form -->
</form>
```

### Tantangan

Tambahkan atribut `action` ke form yang tersedia dengan isian: `https://freecatphotoapp.com/submit-cat-photo`
