# Mulai Projek

Selamat datang di projek kedua. Di projek kali ini kita akan membuat projek website menu kafe. Berbeda dengan projek sebelumnya, di projek kali ini kamu juga akan belajara bagaimana membuat _website_ kamu terlihat lebih bagus lagi.

Untuk memulai projek ini, mari kita mulai dengan membuat HTML sederhana seperti yang kita buat di projek A sebelumnya. Jika kamu belum menyelesaikan projek sebelumnya, selesaikan dulu, ya! üòÅ

### Tantangan

- Buat tag `<html>` baru dengan nilai _lang_ menjadi: "**id**";
- Tambahkan tag `<!DOCTYPE html>`;
- Buat `<title>` di dalam `<head>` dengan isi: "**Menu Kafe**";
- Buat [`<meta>`](https://www.w3schools.com/tags/tag_meta.asp) di dalam `<head>` dengan atribut `charset`, dengan nilai: "**UTF-8**";
- Buat bagian `<main>` di dalam sebuah `<body>`;
- Di dalam `<body>` yang telah kamu buat, tambahkan `<h1>` dengan isi: "**KAFE PENDAKI**";
- Tambahkan juga `<p>` dengan isi: "**Sejak 2023**";
- Nantinya, akan ada dua _section_ dalam website yang kita buat, untuk _section_ untuk kopi, dan camilan. Buat satu `<section>` untuk kopi dalam `<main>`;
- Buat `<h2>` dalam `<section>` dengan nilai: "**Kopi**";
