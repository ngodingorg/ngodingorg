# Atribut

Dalam suatu tag HTML, atribut adalah kata-kata spesial yang ditempatkan di **tag pembuka** untuk mengontrol tag tersebut. Nah, di `<img>` tag sebelumnya, kita bisa memberi tahu komputer sumber gambar yang ingin kita tampilkan melalui atribut `src`.

### Contoh

```html
<img src="https://contoh.com/ini-contoh-url-gambar.jpg" />
```

### Tantangan

Tampilkan gambar dari link: `https://i.imgur.com/JWOiJ0X.jpg`
