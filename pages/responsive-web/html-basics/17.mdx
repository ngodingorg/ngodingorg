# Tag `figcaption`

Setelah menambahkan tag `<figure>` ke gambar yang inginkan, kita sekarang dapat menambahkan deskripsi ke gambar kita.

Untuk menambahkan deskripsi/_caption_ ke gambar yang kita inginkan, kita dapat menambahkan tag `<figcaption>` di bawah gambar yang berada di dalam tag `<figure>` sebelumnya.

### Contoh

```html
<figure>
  <img
    src="https://contoh.com/ini-contoh-link-gambar.jpg"
    alt="Ini contoh gambar"
  />
  <figcaption>Ini contoh deskrpsi gambar</figcaption>
</figure>
```

### Tantangan

Tambahkan deskripsi "**Kucingku suka susu**" ke gambar susu yang ada.
