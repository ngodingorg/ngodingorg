# Tab Baru

Sering kali, kita ingin membuka link di tab baru. Untuk mencapai hal itu, kita bisa menambahkan atribut `target` di tag `<a>` yang kita buat dan menambahkan isian `_blank`.

### Contoh

```html
<a href="https://ngoding.org" target="_blank">Buka ngoding.org</a>
```
