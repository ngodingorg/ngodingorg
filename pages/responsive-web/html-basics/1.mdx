# Tag HTML

Sama halnya dengan daftar isi sebuah buku, pidato, atau bahkan sebuah surat, _website_ yang kamu lihat sehari-hari juga memiliki struktur standarnya tersendiri.

Struktur dalam sebuah _website_ dibuat dalam sebuah bahasa yang dikenal sebagai HTML.

Setiap elemen di dalam bahasa HTML memiliki awalan (misal: `<h1>`) dan akhiran (misal: `</h1>`) yang disebut sebagai **tag**. Sebagai bahan bacaan kamu bisa lihat daftar tag di bahasa HTML di [sini](https://www.w3schools.com/tags/default.asp).

### Contoh

```html
<html>
  <body>
    <!-- Kalimat "Selamat datang!" dibungkus dalam <h1> dan </h1> -->
    <h1>Selamat datang!</h1>
  </body>
</html>
```

### Tantangan
