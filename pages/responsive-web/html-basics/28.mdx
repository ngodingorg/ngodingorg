# Tipe Tombol

Sama halnya dengan tag `<input>`, tag `<button>` merupakan tag multifungsi. Tag tersebut dapat mengirimkan data formulir, atau bisa juga me-_reset_ data. Kamu dapat menjelajahi semua tipe `button` di [sini](https://www.w3schools.com/tags/att_button_type.asp).

Untuk menentukan tipe `button`, kita bisa menggunakan atribut `type`.

### Contoh

```html
<button type="submit">Tombolku</button>
```

### Tantangan

Tambahkan atribut `type` dengan value `submit` ke tag `<button>` yang tersedia.
