# Variasi teks

Mari buat _website_ kita lebih menarik lagi. Tahukan kamu, kamu bisa membuat teks yang ada menjadi **cetak tebal**, <em>miring</em>, atau <u>bergaris bawah</u>. Kamu bisa menggunakan tag-tag berikut:

- `<strong>`: (_bold_) untuk **cetak tebal**
- `<em>`: (_emphasis_) untuk <em>cetak miring</em>
- `<u>`: (_underline_) untuk <u>bergaris bawah</u>

### Contoh

```html
<p>
  Teks ini sangat bervariasi, ada yang <b>tebal</b>, <em>miring</em>, dan
  <u>bergaris bawah</u>.
</p>
```

### Tantangan

Buat kata "_suka_" di deskripsi gambar susu menjadi huruf miring.
