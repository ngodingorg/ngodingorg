# Tag `head`

Lagi-lagi, jika kamu ingat di bagian [Tag `main`](/responsive-web/learn-html-by-building-a-cat-photo-app/a5-main-tag), kita sempat menyinggung terkait pemisahan data/konten _website_ itu sendiri dengan informasi/metadata tambahan lainnya.

Nah, layaknya informasi dalam sampul buku, kita bisa menambahkan berbagai macam informasi terkait website yang kita buat. Misalnya judul halaman yang tampil di _browser_, [favicon](https://id.wikipedia.org/wiki/Favicon), dan lain-lain.

![Contoh _website_ title](https://i.imgur.com/cWBE40t.png)

### Contoh

```html
<html>
  <head>
    <!-- Konten metadata -->
  </head>
  <body>
    <main>
      <!-- Konten utama -->
    </main>

    <footer>
      <!-- Isi Footer -->
    <footer>
  </body>
</html>
```

### Tantangan

Tambahkan `<head>`. Lalu di dalamnya, tambahkan `<title>` yang membungkus: "**Aplikasi Gambar Kucingku**".
