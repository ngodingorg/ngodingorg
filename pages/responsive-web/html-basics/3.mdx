# _Paragraph_ Tag

Hmmm, kalau tadi tag `h1` hingga `h6` merepresentasikan sebuah judul, lalu bagaimana kita merepresentasikan sebuah teks paragraf biasa?

Tenang, [Tim Berners-Lee](https://en.wikipedia.org/wiki/Tim_Berners-Lee), pembuat HTML sudah memikirkan hal itu (kamu bahkan bisa lihat proposal awal HTML di [sini](http://info.cern.ch/hypertext/WWW/MarkUp/Tags.html)). Untuk merepresentasikan suatu teks biasa (teks body), kamu bisa menggunakan _paragraph_ tag (`<p>`).

### Contoh

```html
<p>Aku adalah seorang koders!</p>
```

### Tantangan

Tambahkan tag `p` berisikan "**Lihat lebih banyak gambar kucingku di galeri.**"
