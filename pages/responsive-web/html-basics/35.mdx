# Atribut Kondisi

Sering kali, kita perlu untuk mencegah pengguna memilih suatu input, dan sering kali kita ingin membuat suatu pilihan (`type="radio"` atau `type="checkbox"`) terpilih secara default.

Untuk membuat pilihan tidak dapat diubah, kita bisa menggunakan atribut `disabled`. Sedangkan, untuk membuat suatu pilihan terpilih secara otomatis, kita bisa menggunakan atribut `checked`. Kita juga mengkombinasikan keduanya juga, lho.

### Contoh

```html
<!-- Tidak dapat diubah -->
<input type="checkbox" disabled />

<!-- Opsi terpilih secara default -->
<input type="checkbox" checked />

<!-- Opsi terpilih secara default, dan tidak dapat diubah -->
<input type="checkbox" checked disabled />

<button id="ini-tombol">Tombol</button>
```
