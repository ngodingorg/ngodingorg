# Tag `form`

Luar biasa, kamu sudah mencapai sejauh ini. Sekarang mari kita lanjutkan.

Sekarang, mari kita buat persatuan kucing di daerah rumah kamu. Untuk setiap kucing yang mau bergabung, mereka perlu mengisi formulir pendaftaran terlebih dahulu.

Di HTML, kita dapat menggunakan tag `<form>` untuk mengatur semua kebutuhan formulir dan masukan dari user. Namun sebelum itu, mari kita buat `<section>` baru.

### Contoh

```html
<form>
  <!-- Isian form -->
</form>
```

### Tantangan

Tambahkan `<section>` baru di bawah `section` yang sudah ada. Di dalamnya, tambahkan `h2` dengan isi: "**Formulir Pendaftaran Kucing**". Lalu, tambahkan tag `<form>` di bawah `h2` tersebut (kita biaarkan isinya kosong untuk sekarang).
