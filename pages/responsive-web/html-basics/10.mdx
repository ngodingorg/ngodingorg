# Tag Bersusun

Kalau kamu perhatikan susunan tag `<main>` dan tag-tag lainnya, kamu akan sadar bahwa kamu bisa meletakkan hampir semua jenis tag di dalam tag lainnya. Hal tersebut memungkinkan kiat untuk meletakkan _anchor_ tag/link di dalam suatu kalimat.

### Contoh

```html
<p>
  Kamu bisa belajar pemrograman di
  <a href="https://ngoding.org">Buka ngoding.org</a>
</p>
```

### Tantangan

Ubah link "**Buka Galeri**" dan letakkan di kalimat "**galeri**".
