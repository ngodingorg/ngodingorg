# Tag `footer`

Kalau kamu ingat di bagian [Tag `main`](/responsive-web/learn-html-by-building-a-cat-photo-app/a5-main-tag), kita memisalkan tag `<main>` dengan semua konten di dalam batas margin kertas.

Nah, layaknya catatan kaki di dalam kertas, kita bisa membuat _footer_ di dalam _website_ kita tentu saja dengan tag `<footer>`

### Contoh

```html
<html>
  <body>
    <main>
      <!-- Konten utama -->
    </main>

    <footer>
      <!-- Isi Footer -->
    <footer>
  </body>
</html>
```

### Tantangan

Tambahkan `<footer>`. Lalu di dalamnya, tambahkan _paragraph_ tag dengan isi: "**Tanpa Hak Cipta - ngoding.org**". Buat teks `ngoding.org` menjadi link yang membuka tab baru ke `https://ngoding.org`.
