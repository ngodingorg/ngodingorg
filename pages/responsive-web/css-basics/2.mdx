# Tag `style`

Kita dapat mengatur tampilan kita dengan menggunakan [CSS](https://id.wikipedia.org/wiki/Cascading_Style_Sheets) (_Cascading Style Sheets_). Terdapat banyak cara untuk menggunakan CSS, salah satunya dengan menggunakan tag `<style>` di dalam `<head>` kita.

Dengan menggunakan tag `<style>` kita dapat mengatur tamoilan untuk semua komponen yang ada di file HTML kita. Kedepannya, kita akan belajar bagimana mengatur kode CSS kita agar dapat digunakan berulang kali di beberapa file HTML sekaligus.

### Contoh

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <style>
      /* Kode CSS kita */
    </style>
  </head>
</html>
```

### Tantangan

Tambahkan tag `<style>` di dalam `<head>` yang tersedia. Kita biarkan isinya kosong untuk sekarang.
