# Tag `button`

Agar pengguna dapat mengirimkan data yang telah mereka isikan, kita dapat menambahkan tombol yang pengguna bisa tekan.

Seperti yang sudah kalian duga, kita bisa menggunakan tag yang disediakan. Tag tersebut adalah tag `<buttton>`. Tag `<button>` memiliki banyak kemiripan dengan tag `<a>`. Dan seperti yang kalian duga, kita bisa menambahkan tag lain di dalam tag `<button>` untuk membuatnya dapat ditekan.

### Contoh

```html
<button>Tombol</button>
```

### Tantangan

Tambahkan tag `<button>` dengan judul "**Kirim Data**" di bawah `input` yang tersedia.
