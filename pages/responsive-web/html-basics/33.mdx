# Tag `legend`

Kita sudah belajar bahwa tag `<fieldset>` dapat digunakan untuk mengelompokkan beberapa input sekaligus. Lalu, bagaimana kita memberi tau pengguna apa tujuan sekelompok _input_ tersebut?

Layaknya sebuah peta, kita bisa memberikan penanda dalam suatu `<fieldset>` dengan menggunakan tag `<legend>`.

### Contoh

```html
<fieldset>
  <legend>Masukkan item-item anda</legend>

  <input type="radio" id="opsi-item-1" value="item1" /><label>Item #1</label> j
  <input type="text" id="input-teks" name="input1" />
</fieldset>
```

### Tantangan

Buat `<legend>` di dalam `<fieldset>` yang tersedia dengan isi: "**Apakah kucing kamu suka main di dalam atau di luar?**"
