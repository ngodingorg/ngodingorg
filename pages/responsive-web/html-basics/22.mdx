# Tag `input`

Mari kita isi formulir kita. Pertama, kita ingin mendapatkan isian melalui teks (_text input_).

Di, HTML kita bisa menggunakan tag `<input>` untuk mendapatkan isian dari pengguna. Sama halnya dengan tag `<img>`, tag `<input>` juga berjenis _self-closing tag_.

### Contoh

```html
<input />
```

### Tantangan

Tambahkan tag `<input>` ke dalam `<form>` yang tersedia.
