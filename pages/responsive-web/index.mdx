# Desain Web Responsif

Di modul Desain Web Responsif ini, kalian akan belajar tentang bahasa pemrograman yang digunakan programmer profesional untuk membuat halaman web (_website_): [HTML](https://id.wikipedia.org/wiki/HTML) (_Hypertext Markup Language_). Tidak hanya itu, kalian juga akan belajar bagaimana mempercantik tampilan _website_ yang kalian buat dengan [CSS](https://id.wikipedia.org/wiki/Cascading_Style_Sheets) (_Cascading Style Sheet_).

Di modul ini, kalian akan membuat website foto kucing ataupun hewan kesayangan kalian. Kemudian, kalian akan belajar tentang topik yang digunakan progammer profesional, seperti [variabel CSS](https://www.w3schools.com/css/css3_variables.asp) dengan membuat gambar pinguin. Selain itu, kalian juga akan belajar tentang aksesibilitas _website_ dengan membuat aplikasi kuis sederhana.

Tidak lupa, kalian juga akan belajar membuat _website_ yang dapat menyesuaikan bentuknya sesuai dengan ukuran layar dengan membuat galeri foto kamu sendiri.
