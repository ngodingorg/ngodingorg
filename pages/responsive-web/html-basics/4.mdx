# _Comment_

Kalau kamu perhatikan pada materi sebelumnya, kami meninggalkan petunjuk untuk kamu agar memudahkan kamu menyelesaiakan tantangan yanga ada sebelumnya dalam bentuk catatan. Misalnya:

```html
<html>
  <body>
    <!-- Kalimat 'Selamat datang!' dibungkus dalam <h1> dan </h1> -->
    <h1>Selamat datang!</h1>
  </body>
</html>
```

Mungkin kamu bertanya-tanya, kenapa kalimat: '_Kalimat "Selamat datang!" dibungkus dalam `<h1>` dan `</h1>`_' tidak tampil di _website_ yang kamu buat?

Hal itu karena sistem komputer mengabaikan baris kode itu. Catatan dalam sebuah kode komputer sering disebut sebagai **Comment**. Setiap bahasa pemrograman, punya cara untuk membuat komen mereka masing-masing. Dalam HTML, teks komentar dibungkus dalam `<!--` dan `-->`.

<br />

<img src="https://i.imgur.com/pGNLZm8.png" />

### Tantangan

Tambahkan _comment_ di atas tag `h1` berisikan: "**Jangan lupa hapus h1**".
